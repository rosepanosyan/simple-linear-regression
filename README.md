# Simple Linear Regression: Age vs Cholesterol

This project demonstrates a simple linear regression model using Python and Scikit-learn.

The goal of the project is to analyze the relationship between age and cholesterol level using regression analysis and visualization techniques.

## Technologies Used

- Python
- NumPy
- Matplotlib
- Scikit-learn

## Dataset

The dataset contains:

- **Independent Variable (X):** Age
- **Dependent Variable (Y):** Cholesterol Level

## Features

- Linear regression model fitting
- Regression line visualization
- Residual plot visualization
- Model evaluation metrics:
  - Mean Squared Error (MSE)
  - Root Mean Squared Error (RMSE)
  - R² Score

## Project Workflow

1. Import required libraries
2. Create and prepare the dataset
3. Train the linear regression model
4. Predict target values
5. Visualize the regression line
6. Analyze residuals
7. Evaluate model performance
